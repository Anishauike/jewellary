# Jewellery Store Website

A responsive and visually appealing jewellery store website built using **HTML, CSS, and JavaScript**. This project was created as my first web development project to learn the fundamentals of frontend development, including page layouts, navigation, styling, and basic user interactions.

---

## Overview

The Jewellery Store Website showcases a collection of elegant jewellery products through a clean and user-friendly interface. It includes multiple webpages such as Home, About, Products, Contact, Login, and Shopping Cart, providing a complete static e-commerce website experience.

This project helped me understand the basics of responsive web design, HTML structure, CSS styling, and JavaScript functionality.

---

## Features

* Attractive homepage with featured jewellery collections
* Product listing page
* Shopping cart interface
* User login page
* Contact page
* About page
* Responsive layout
* Interactive hover effects and animations
* Easy navigation between pages
